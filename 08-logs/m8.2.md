## Task 1 — Temporal engine (symbolic schema · resolver · tokens · render)

**The pure-logic core of "LLMs classify, code computes"** ([ADR-056](../adr/056-temporal-correctness-date-tokens.md),
CLAUDE.md hard rule 12). First task in the milestone; **no migration, no DB, no LLM, no wiring** —
all downstream consumers (organizer T2, indexer/chat/MCP/capsule T3, web mirror T4) build on this.
New package `server/app/temporal/`. **Committed locally (`d30d6b6` + review fixes `2787701`), not
pushed.**

### What was built

- **`symbolic.py` — the emission contract.** A fail-closed pydantic discriminated union (on `kind`)
  of exactly what the organizer LLM may emit — never a computed date, only a *classification*:
  `ExplicitRef` (whole/partial calendar date, year optional → code snaps), `RelativeRef`
  (`unit` day/week/month/year × signed `offset`), `WeekdayRef` (weekday × last/this/next),
  `MonthRef` (month × year|direction), `SeasonRef` (season × year|year_offset). `extra="forbid"`
  guards against silent drift; validators coerce common variants ("Tuesday"→`tue`, "fall"→`autumn`).
  `parse_symbolic(data)` returns `None` on anything invalid — **never raises** on bad model output.
- **`resolver.py` — the deterministic compute.** `resolve(ref, anchor)` / `resolve_reference(data,
  anchor)` turn a symbolic ref into an absolute `ResolvedTime` against the capture's **stored
  anchor** (never wall-clock — reprocess-determinism, ADR-056 §1): offset arithmetic, weekday walks
  (last/this/next, `or 7` so same-weekday goes back a full week), month/year snapping, yearless
  snap to the most-recent-past occurrence (year-by-year back-walk, 8-year reach covers a century
  leap gap), northern-hemisphere season windows (winter spans the year boundary). Anything it can't
  compute to a real date → `None` (fail-closed, rule 12). Pure stdlib (`datetime`/`calendar` +
  integer month math), **no `dateutil`** — so the T4 web mirror computes byte-identical results.
- **`tokens.py` — the `[[t:START[/END][|label]]]` token + `PartialDate`.** Partial-ISO at honest
  granularity (`2025` / `2025-07` / `2025-07-07` / `2025-07-07T22:00`); parse/serialize, `find_tokens`
  locates every token in a body (malformed → `resolved=None`, surfaced not crashed), and the
  **day-granular `occurred` floor/ceil** the DB stores (`occurred_*` stay `date` — tokens own sub-day,
  ADR-056 §6; a coarse single partial honestly spans its period for the ADR-049 dedup gate, a
  precise day/minute is a point → `occurred_end` None).
- **`render.py` — tokens → text, never shown raw** (ADR-056 §4). `render_relative` = the live phrase
  ("10 days ago", "last month", "summer 2025") computed at *now*, with `render_absolute` as the
  tooltip; `expand_for_index` = stable absolute only (vectors see language, not token noise);
  `expand_for_llm` = absolute + a fresh relative hint ("7 July 2026 (10 days ago)") for the LLM-bound
  rendering contract. Whole-body helpers replace every token. The humanization spec is documented for
  the web mirror; **rounding pinned to round-half-up** (`math.floor(x+0.5)`) so JS `Math.round` agrees
  at `.5` ties.

### Tests, lint, review

- **57 unit tests, zero mocks** (`tests/test_temporal.py`) — schema validation/fail-closed, every
  resolver branch against a fixed **Friday** anchor (2026-07-17), token round-trips + `occurred`
  mapping, and all four render audiences. Both ADR-056 Accept scenarios exercised: "10 days ago" →
  "10 days ago" today / "a year ago" next year (tooltip exact date); "last summer" →
  `[[t:2025-06/2025-08|summer 2025]]`. **Full suite 882 green, ruff + format clean.**
- **Independent review** (fresh agent, ADR-056 + 08 §M8.2 + CLAUDE.md + diff): **CHANGES-REQUIRED**
  → **1 must-fix + 2 minors, all resolved** (`2787701`).
  - **MUST-FIX (resolved):** yearless "29 Feb" against a leap-year anchor fell through to a *future*
    date when the `y-1` candidate was invalid (`stepped or candidate` kept the known-future date).
    Rewrote yearless snapping to the back-walk described above → previous real leap year (2020), or
    `None` when no valid past occurrence is in reach. Regression-tested.
  - **MINOR (resolved):** `round()` half-to-even vs JS `Math.round` half-up would diverge at `.5`
    ties in the not-yet-built web mirror → pinned to round-half-up (`_round`), documented in the spec.
  - **MINOR (resolved):** test-coverage adds — Feb-29 leap boundary, impossible-day→None, round-half-up
    ties, future coarse (month/year) rendering, mixed-granularity range round-trip.
  - The reviewer independently re-verified all resolver math, weekday walks, season windows, token
    round-trips, both Accept scenarios, web-portability, and rule-12 fail-closed discipline as correct.
